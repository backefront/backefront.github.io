Como Funciona O Select Sort

Você sabe como funciona o método Select Sort? Ele é um algoritmo baseado em passar sempre o menor valor do vetor para a primeira posição, repetindo este passo até que o vetor esteja totalmente organizado. Então, como podemos fazer isso? O Felipe Casseb explica nesta aula como funciona este método de ordenação!
